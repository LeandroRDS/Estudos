Tags, para que serve e como usalas

Tag = <h1>.
Função = serve para colocar um texto como: titulo de primeiro grau.
Modo de uso = Para usar é bem simpler basta abrir e fechar a tag ex <h1> Titulo Exemplo </h1>.

Tag = < < > e < > >.
Função = serve para representar os caracters menor que < e maior que >. Modo de uso = basta escrever < e em seguida clocar um ; para ativar a tag.
Tag = &#x. Função = Adicionar emoji. Modo de uso = escreva essecodigo &#x e incira o codigo do emoji que você quer, o codigo é pego
