Como Funciona a HTML e as CSSs

As CSSs são chamadas no genero femenino por ser a sigla de "as folhas de estilo em cascata"
tradusido do ingles the cascading style sheets e HTML e tradado como genero femenino por ser a linguagem HTML
(HyperText MarKup Language ou linguagem de marcação de hipertexto). A HTML e as CSSs não são linguagens de
programação, elas são linguagens de marcação (por isso é errado dizer que você programa em HTML, o certo é
dizer qeu você desenvolve em HTML e CSS ). o HTML é focado no conteúdo da pagina, toda a informação que aparece na pagina esta em HTML como os textos e
as imagens, já as CSSs são focadas na aparencia (dsing) da pagina, pra ficar mais facil de entender, as CSSs é
tudo aquilo que deixa a pagina bunitinha ksksksk. Ainda tem uma Terceira linguagem que vamos abordar que que é
o Java Script JS (Ir para Java Script).
