Como Funciona a HTML e as CSSs

As CSSs são chamadas no genero femenino por ser a sigla de "as folhas de estilo em cascata"
tradusido do ingles the cascading style sheets e HTML e tradado como genero femenino por ser a linguagem HTML
(HyperText MarKup Language ou linguagem de marcação de hipertexto). A HTML e as CSSs não são linguagens de
programação, elas são linguagens de marcação (por isso é errado dizer que você programa em HTML, o certo é
dizer qeu você desenvolve em HTML e CSS ).
o HTML é focado no conteúdo da pagina, toda a informação que aparece na pagina esta em HTML como os textos e
as imagens, já as CSSs são focadas na aparencia (dsing) da pagina, pra ficar mais facil de entender, as CSSs é
tudo aquilo que deixa a pagina bunitinha ksksksk. Ainda tem uma Terceira linguagem que vamos abordar que que é
o Java Script JS (Ir para Java Script).
No HTML a maioria das tags tem abertura (<h1>) e fechamento (</h1>), mas nem todas tem abertura de
fechamento, como por exemplo a tag <img src="foto.png" alt="Exemplo de foto"> mas podemos ter em todas as
tags parâmetros, nesa tag temos o parâmetro src e o alt e em cada parâmetro temos o seu volar.
Nas CSS usamos seletores por exemplo, h1{ } isso é um seletor da tag h1 e dentro do seletor vamos colocar
linhas, uma linha inteira é chamada de declaração e cada declaração é um par de propriedades e valores.

A Estrutura Basica do HTML 5

A primeira linha em um arquivo de html deve ser para indicar ao navegador que esse arquivo é um html
<!DCOTYPE html>, depois colocamos no arquivo html a tag que indica o inicio do arquivo em sí
<html lang="pt-br"> </html>, nessa mesma linha temos o parâmetro lang e seu valot pt-br
que serve para definir a linguagem padrão desse site. Na estrutura basica do html temos o <head>
</head> e o <body> </body>. A área head é para as configurações do site e a área
é o corpo do site. Algumas configurações basicas são <meta charset="UTF-8"> para que o navegador
interprete corretamente os acentos e pontuações, temos tambem <meta="viewport" content="width=device-width
, initial-scale=1.0"> essa propriedade serve para indicar ao navegardor que o site vai abrir em
uma escala de 100% e vai ocupar todo o espaço da tela, temos tambem o <title> que serve para colocar
o titulo do site.
